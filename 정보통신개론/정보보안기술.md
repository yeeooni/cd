# 정보 보안 기술

1. 정보 보안 기술의 개념
- 컴퓨터 범죄를 억제하고 정보 자산을 보호하기 위한 기술 및 시스템
- **암호화(Encryption)** 기술 : 암호(Cipher)를 사용해 평문(Paintext)을 암호문으로 변환하는 것
- 복호화(Decryption) : 암호문을 원래의 평문으로 복원하는 것
- 인증 기술 : 컴퓨터로 주고받는 문서에 대한 작성자의 신원을 보증하고 문서 내용을 인증하는데 사용되는 기술
- 전자 서명 : 전자 문서에 기존의 서명 또는 인감과 동일한 역할을 하는 서명을 하는 것
- 디지털 서명 : 메시지 인증과 사용자 인증을 포함하는 개념
- 공인인증서 : 공인인증기관이 발행하는 전자정보형태의 사이버 거래용 인감증명서
- 방화벽 : 외부의 공격으로부터 시스템을 보호하고 내부의 중요한 정보가 유출되지 않도록 차단하는 하드웨어 및 소프트웨어
