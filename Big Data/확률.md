# 확률 

> 모집단 (전수조사) >> 지표해석 >> 가치  
(표본조사) >> 통계학 >> 지표해석 

* 통계적특성 도출 
  * 평균, 표준편차, 중앙값
  * 최대, 최소 값

* 동전 던지기 실험 - 확률실험 
> 동전 2개에 대한 확률 실험  
표본 공간 = HH HT TT TH (H : 앞 / T : 뒤)

* 주사위를 굴려 홀수(모수)가 나올 확률  
확률공간 : 1 ~ 6  
각 눈이 나올 확률 : 110  
홀수 : 135  
확률 : 3 / 6 = 1 / 2 = 0.5 (50%)

* 확률 분포  
모수[^1]와 통계량 설명   
대표값과 평균에 대한 설명  
확률 변수 

[^1]: 모집단의 관심대상
