1. 설정, 초기화  

   - 전역 사용자명 / 이메일 구성

     - `git config --global user.name "Your Name"`
     - `git config --global user.email "Your Email"`

   - 저장소별 사용자명 / 이메일 구성

     - `git config user.name "Your Name"`
     - `git config user.email "Your Email"`

   - 전역 설정 정보 조회

     - `git config --global --list`

   - 저장소별 설정 정보 조회

     - `git config --list`

   - git 출력결과 색상 활성화

     - `git config --global color.ui "auto"`
